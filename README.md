## Overview

This project is getting venue datas from Google Books API.( https://developers.google.com/books/ )
You can just clone this sample repository and know how it works very easily.

#### MainActivity


Get 5 books from each 6 genres 
 - Science fiction
 - Romance
 - Mystery
 - Horror
 - Travel
 - Art
